# Economics Blind Box Demo Skeleton

Khung sườn Next.js (App Router + TypeScript) theo spec trong ảnh:

- Intro/Start + chọn preset MBTI
- Music Player placeholder
- Gallery/Polaroid placeholder
- Xưa vs Nay placeholder
- Blind Box shelf 4-5 box + tiến độ sticker
- Sticker Album + Final Letter unlock

## Cấu trúc chính

- `src/app`: trang và layout
- `src/components`: component UI (demo shell)
- `src/lib`: type dùng chung
- `data/*.json`: dữ liệu mock để bạn thay sau
- `public/music`, `public/photos/*`: chỗ đặt assets thật

## Chạy local

```bash
npm install
npm run dev
```
